== README

GISS ou Gestão Integrada de Serviços de Saúde, é uma aplicação em ambiente gráfico com o propósito de gerir um serviço de saúde.

== INSTALL

1- Criar pasta no disco C com o nome 'BDGiss' (C:\BDGiss). //pasta onde ficará a base de dados

2- Executar Script Criar_DB.sql //Criar base de dados

3- Executar Script Criar_Tabelas.sql //Criar tabelas da base de dados

4- Executar Script Inserir_Dados.sql //Inserir os dados nas tabelas
